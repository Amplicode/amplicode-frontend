Optional count

count: number
  • Defined in jmix-react-ui/src/crud/list/useEntityList.ts:108

Total number of entity instances.

When EntityListHookOptions.entityList is not used, count will contain the value retrived from backend upon execution of EntityListHookOptions.listQuery (it is also obtainable as listQueryResult.data.${entityName}Count).

When EntityListHookOptions.entityList is used, count will contain the total number of elements in EntityListHookOptions.entityList.

Optional items

items: EntityInstance<TEntity, MayHaveId, MayHaveInstanceName>[]
  • Defined in jmix-react-ui/src/crud/list/useEntityList.ts:97

Entity instances that will be displayed by the list component.

When EntityListHookOptions.entityList is not used, items will contain the entity instances retrived from backend upon execution of EntityListHookOptions.listQuery (it is also obtainable as listQueryResult.data.${entityName}List).

When EntityListHookOptions.entityList is used, items will contain a relevant portion of EntityListHookOptions.entityList depending on pagination / sorting / filtering.
